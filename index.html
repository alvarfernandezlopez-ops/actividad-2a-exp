Datos generales

DATOS GENERALES DE LA SITUACIÓN DE APRENDIZAJE:

- Etapa y nivel: Educación Secundaria Obligatoria ( 3ºESO)

- Materia: Matemáticas

- Temporalización: ( sesiones de 55 minutos.

- Agrupamientos y organización: Se utilizará el trabajo en gran grupo para la explicación, parejas para la toma de medidas en el gimnasio y grupos heterogéneos de cuatro para el presupuesto (siguiendo principios DUA).

PRODUCTOS EXIGIDOS AL ALUMNADO

- Infografía: Resumen visual de áreas, materiales y presupuesto final.

- Podcast: Explicación del proceso y justificación económica del proyecto.

HERRAMIENTAS TECNOLÓGICAS

- Para el profesorado: eXeLearning para la guía docente y Microsoft Teams (Educastur) para gestión.

- Para el alumnado: Hojas de cálculo, Genially para la infografía y Audacity para el podcast.

- Recursos: Calculadoras, cinta métrica y acceso a catálogos online de pinturas.
